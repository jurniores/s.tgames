Fire Shot

version 1.0.0

Fire Shot é um jogo FPS de primeira pessoa, desenvolvido na Unity pela S.T Games. Jogo MultiPlayer de servidor aberto, criado com o intuito de demonstrar as minhas habilidades em jogos online. O Network de rede usado para proporcionar o modo multiplayer foi o Omni, produzido por Ruan Cardoso.

Para poder acessar o jogo, é só fazer o download no botão acima e seguir alguns passos ensinado neste vídeo do youtube >> Como jogar o Fire Shot
